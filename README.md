# Todo API (FastAPI)

This is a simple Todo API built using FastAPI.

## Features
- Add tasks
- View all tasks
- Delete tasks

## Tech Stack
- Python
- FastAPI

## How to Run
1. Install dependencies:
   pip install fastapi uvicorn

2. Run server:
   uvicorn main:app --reload

3. Open in browser:
   http://127.0.0.1:8000/docs

## Description
This project demonstrates basic REST API development with CRUD operations using FastAPI.
